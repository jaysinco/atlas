### Get SSH Host
```
sudo netstat -tnpa | grep sshd
```

### Find Ip or Mac
```
arp-scan --localnet | grep <mac or ip>
nmap -n -sn --send-ip 192.168.3.0/24
ip neigh | grep <mac or ip>
```
